# memcontacts
MemContacts - Web platform to exercise the memory with a virtual contact book

### Tarea 1: Insertar imagen.

   Para insertar una imagen a un contacto que aun no la tenga, podremos hacerlo de varias maneras:
	-Desde la pagina principal pulsando en el enlace "N contactos aún no tienen foto."
	-Desde la agenda pinchando en el cuadro sin imagen
	-Accediendo directamente a la ventana "Actualizar fotos" dentro de Agenda.
	-A partir del juego "NumberMaster", en los enlaces de los contactos sin foto.
  Seleccionaremos el contacto a actualizar, y en la ventana emergente buscamos la imagen que 
asociaremos al contacto. Para poder subir un archivo de imagen hemos habilitado todos los permisos
a la carpeta "imagenes/contacts", dónde se almacenan las fotos de contactos.


### Tarea 2: Juegar a MemContacts

   Para jugar a uno de nuestros juegos, podemos acceder desde el menu de juegos (donde se muestra 
un pequeño resumen de cada juego) o accediendo directamente al juego que queramos desde la barra 
de herramientas.
  Para el primer juego debemos seleccionar el nombre correcto que se corresponda con la foto, en el 
segundo juego lo que tenemos que hacer es adivinar el número de teléfono del contacto.

### Tarea 3: Pregunta del día

   Esta tarea se divide en 2 partes:
	- Realizar pregunta que podremos acceder desde la barra de herramientas o en la pagina
	principal de "Pregunta del día"
	- Responder una pregunta a la que accederemos de la misma manera que la anterior
  Se puede enviar una pregunta a cualquiera de los contactos del usuario, y dicho contacto la podrá 
responder cuando se conecte. De la misma forma, se podrá responder a cualquiera de las preguntas
recibidas de tus contactos.
